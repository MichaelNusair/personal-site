## Preliquify — Open-Source Shopify Framework

### The Problem
Building Shopify theme UI is painful. Shopify's Liquid templating language is limited, developer tooling is primitive, and there's no good way to use modern component frameworks (React, Preact) in Shopify themes while maintaining server-side rendering performance.

### The Solution
**Preliquify** is an open-source framework that compiles **JSX/Preact components** into **Shopify Liquid templates** with client-side hydration.

Write your Shopify UI once in Preact/JSX, and Preliquify generates:
1. **SSR Liquid templates** — Server-rendered HTML that works natively in Shopify themes
2. **CSR hydration scripts** — Client-side JavaScript that makes the components interactive

### Why It Matters
- **Developer experience** — Write modern component-based UI instead of raw Liquid
- **Performance** — Server-side rendered by default, hydrated on the client only when needed
- **Maintainability** — One codebase for both server and client rendering
- **Shopify-native** — Generated Liquid works with Shopify's theme system, not against it

### Origin Story
Preliquify wasn't built as a weekend experiment. It emerged from a real production need at **Tolstoy**, where Michael was building the Media Gallery widget for Shopify merchants. He needed a way to write Preact components that compiled to performant Liquid templates — and when no existing tool could do it, he built one and open-sourced it.
